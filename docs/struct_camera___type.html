Camera 物件定義

為了簡化操作，所以這邊有儲存玩家資料，但真正需要用到角色資料時，可以從外部傳入，因此 player 後續可以拿掉。

start 跟 end 這兩個資料後續可以用 Rectangle 或 Two_Point 處理。
