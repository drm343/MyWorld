MyWorld
Create your own roguelike in C

本專案使用 C 撰寫，需要安裝 SDL2、SDL2-ttf、libconfig，如果你是 windows 的使用者，請考慮使用 "Bash on Ubuntu"。

下列工具為選擇性的，可根據需求選擇是否安裝。
doxygen、graphviz、git、make、indent、blogc。

目前自動化建構可以從 make 或 tup 中選一個來使用。
